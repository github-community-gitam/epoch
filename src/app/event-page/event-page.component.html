Epoch 1.0 is the first large-scale tech-related event that GITAM University experiences. It is a two-day fest conducted by GCG at GITAM University, Bangalore. This fest primarily focuses on promoting technology among students in GITAM.

There are four main events - Technical Treasure Hunt, LAN Gaming, Product Commercial Videography, and Tech Trivia. These four events are conducted on all three campuses before the fest. Finalists in each event from each campus shall come to the Bangalore campus to compete with the other finalists from the other campuses.
